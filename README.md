# EDA-Student-Performance-Analysis
Exploratory Data Analysis of a student performance dataset using Python, Pandas, Matplotlib, and Seaborn.
# Exploratory Data Analysis (EDA) Project

## Overview

This project performs Exploratory Data Analysis (EDA) on a student performance dataset using Python.

## Features

* Data loading and inspection
* Missing value analysis
* Statistical summaries
* Histograms
* Boxplots
* Correlation heatmap
* Pairplot visualization
* Outlier detection
* Automated report generation

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Structure

EDA_Project/
├── data/
├── images/
├── reports/
├── eda.py
├── requirements.txt
└── README.md

## Installation

pip install -r requirements.txt

## Run

python eda.py

## Output

Generated visualizations are saved in the `images/` folder and the report is saved in the `reports/` folder.
